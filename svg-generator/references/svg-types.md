# SVG 图型选择参考

## 1. 封面图

适合：

- 文章头图
- 系列首图
- 社媒分享图

目标：

- 第一眼传达主题
- 停留 1-2 秒内就知道这讲什么

不要：

- 塞太多正文
- 放太多次要元素

## 2. 对比图

适合：

- 新旧方案对比
- 错误做法 vs 正确做法
- Before / After

目标：

- 让差异一眼可见

做法：

- 左右分栏
- 用颜色、图标、对齐强化对比关系

## 3. 流程图

适合：

- 步骤说明
- 因果关系
- 状态流转

目标：

- 让人知道先后顺序和依赖关系

做法：

- 从上到下或从左到右保持单一阅读方向
- 箭头只表达一种主要流向

## 4. 框架图

适合：

- 模型、方法论、系统结构
- 模块关系
- 分层架构

目标：

- 让抽象概念有边界和层次

做法：

- 先画容器关系，再填文字
- 层级不要超过 3 层，超出就拆图

## 5. 金句卡

适合：

- 传播一句话观点
- 文章结尾卡片

目标：

- 强记忆点
- 易转发

做法：

- 大字
- 大留白
- 极少装饰

## 6. 轻插画 / Hero Graphic

适合：

- 网页头图
- 产品介绍页
- 文档配图

目标：

- 建立气质和主题氛围

做法：

- 以几何形、少量拟物、少量动势为主
- 不要为了“像插画”牺牲信息清晰度
