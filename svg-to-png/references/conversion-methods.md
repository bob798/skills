# SVG 转 PNG 方法参考

## 1. qlmanage

适合：

- macOS 本机快速导出
- 不想额外安装工具

特点：

- 系统自带
- 适合本地预览和轻量导出

限制：

- 参数能力弱于专业渲染工具

## 2. rsvg-convert

适合：

- 命令行稳定批量转换
- CI 或脚本化流程

特点：

- 快
- 输出稳定
- 适合纯图形和普通文本

## 3. Inkscape

适合：

- SVG 比较复杂
- 对兼容性要求高

特点：

- 渲染完整度通常更好
- 命令行参数适合导出

## 4. ImageMagick

适合：

- 环境里已经有 `magick`
- 只是做常规栅格化

限制：

- 不同机器的 SVG delegate 能力差异较大

## 5. 浏览器 / 设计工具导出

只在以下情况使用：

- 命令行工具都没有
- 用户只要一次性结果，不要求可复现

缺点：

- 不适合批量
- 不够稳定
- 容易受窗口缩放和截图质量影响
