# article-writer

将个人经验转化为高质量文章的 Claude Skill。

## 包含内容
- `SKILL.md` — 主文件，Claude 读取
- `references/writer-patterns.md` — 三种写作模式（Gladwell/PG/Thompson）
- `references/theory-bank.md` — 学术理论背书速查

## 适用场景
公众号长文 · 思想型文章 · 系列内容规划

## 快速开始
把 SKILL.md 安装到 Claude 后，直接说：
「帮我写一篇关于 [主题] 的文章」
Claude 会先做采集访谈，再开始写。
