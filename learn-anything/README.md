# Learn Anything Skill

> Structure your learning. Build the shortest path from "I want to learn X" to "I built something with X."

Most learning fails not because of lack of resources — but because of lack of structure. This skill cuts through resource overload and gives you a concrete, personalized learning path.

---

## When to Use

- "How do I get started with X?"
- "Give me a learning plan for X"
- "What's the best way to learn X as a beginner?"
- "I've been learning X but feel lost"
- "What resources should I use for X?"

---

## What It Does

1. **Diagnoses** your current level and learning context
2. **Maps** the full landscape (so you know where you are)
3. **Finds** the shortest path to your first real output
4. **Recommends** 3-5 high-quality resources (not a list of 50)
5. **Gives** you one concrete task to do today

---

## Core Philosophy

```
Learning is complete when you can BUILD something, not when you've READ something.
The best learning plan is the one that gets you to a working demo fastest.
```
